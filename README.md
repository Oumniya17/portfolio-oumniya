## 🚀 Run locally

```bash
# Clone the repo
git clone https://github.com/Oumniya17/portfolio-oumniya.git

# Enter the project
cd portfolio-oumniya

# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm run test

# Build for production
npm run build
```
